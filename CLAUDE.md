## После каждого ответа выводи:
◇ ctx: [used]K | [left]K | [цвет][X%/100%] | NORMAL
◇ eval: T:[✓/✗] R:[✓/✗] C:[✓/✗]

### Правила цвета:
- 0-50%   → 🟢
- 50-80%  → 🟡
- 80-100% → 🔴

### Расшифровка полей:
# ctx used  = токены потрачены в этой сессии
# ctx left  = осталось до конца окна (макс ~200K для Sonnet)
# eval T    = Tools  — могу запускать bash
# eval R    = Read   — могу читать файлы  
# eval C    = Code   — могу писать/выполнять код
