# AGENTS — правила для агентов в MAUOS

Краткий контракт для Claude Code, Codex, Cursor и аналогов. Детали концепции — в `README.md`.

## 1. Границы

- Работай в рамках задачи пользователя; не расширяй scope без причины.
- Не публикуй и не копируй в ответы содержимое файлов, помеченных как приватные пользователем (если такие есть вне этого репо).

## 2. Файлы и структура

- **File‑first:** правда в заметках и артефактах на диске, не «только в чате».
- Новые заметки — по шаблонам из `_templates/` и канону имён из `00_core/MAUOS_Obsidian_Plan.md` (или локальных правил в vault).
- Индексы в `99_indexes/` поддерживай актуальными, когда пользователь просит реорганизацию.

## 3. Безопасность

- Секреты только в `.env` или вне репозитория; в Git — только `.env.example` без реальных значений.
- В публичные коммиты не включать: ключи, токены, личные дневники сессий, абсолютные пути к домашнему каталогу других людей.

## 4. Git

- Осмысленные сообщения коммитов; не коммить мусор и локальные артефакты IDE.
- Не выполняй разрушающие операции (`git reset --hard`, force push) без явного запроса.

## 5. Вдохновение

Паттерн публичного шаблона близок к [CybOS](https://github.com/Gerstep/cybos): в открытом репозитории — скелет и документация, приватное — у владельца vault.
